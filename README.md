# 🏛️ Jura Parlement

Suivi des mentions du canton du Jura et de la péréquation financière dans les objets et débats parlementaires suisses.

## Mots-clés recherchés

- **Jura**, **Moutier**
- **Charles Juillard**, **Mathilde Crevoisier Crelier** (Conseil des États)
- **Thomas Stettler**, **Loïc Dobler** (Conseil national)
- **RPT**, **péréquation financière**, **Finanzausgleich**, **NFA**

## Structure

- `home.html` : Page d'accueil avec résumé de session
- `objects.html` : Liste des objets parlementaires
- `debates.html` : Interventions orales dans les débats
- `stats.html` : Statistiques

## Scripts R

- `Recherche_Jura.R` : Recherche des objets parlementaires
- `Recherche_Debats_Jura.R` : Recherche dans les débats

## Couverture

Législature 52 uniquement (dès décembre 2023).

## Mise à jour automatique

GitHub Actions met à jour les données 2x/jour (13h et 22h, heure suisse).
